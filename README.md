## 技术栈
**ES6**: 采用ES6语法，需要babel  
**HTML5**: 拖放，文件，canvas等HTML5的API  
**Webpack**: 自动化构建工具，刚接触，并配置的开发和生产两套配置。热加载不够完善  
## 使用
````bash
# 安装依赖
npm install
# 开发环境 默认访问地址localhost:8080
npm run start
# 生产 dist/
npm run build
````
